# Evolutionary Mini Project


## Development

Requires `poetry` installed (tested with `v1.2.2`).

Install the dependencies:

```sh
poetry install
```

Spawn a shell within the project's environment:

```
poetry shell
```


## Unit Tests

To run the unit-tests, use:

```
pytest
```


## Formatting

To run formatting, use:

```
black evokity tests
```
